Built for Developers

Clean APIs that get out of your way. Full TypeScript inference, zero config required.

retry-example.ts
import { retry, CircuitBreaker } from '@corvid-agent/retry';

// Simple retry with backoff
const data = await retry( () => fetch('https://api.example.com'), { maxAttempts: 3, backoff: 'exponential' } );

// Circuit breaker pattern
const breaker = new CircuitBreaker({ threshold: 5, resetTimeout: 30_000 });

const result = await breaker.execute( () => riskyOperation() );

Resilient by Default

Handle transient failures gracefully with battle-tested retry patterns. Exponential backoff with jitter prevents thundering herds. Circuit breakers protect downstream services.

  • Exponential, linear, and fixed backoff strategies
  • Configurable jitter for distributed systems
  • Circuit breaker with half-open recovery
  • Full abort signal support


Compose Everything

Build complex transformations from simple functions. Full type inference flows through the entire pipeline. Async and sync pipes work seamlessly together.

  • Type-safe pipelines up to 12 stages
  • Async pipeline support
  • Function composition (right-to-left)
  • Tap for side effects without breaking flow
pipe-example.ts
import { pipe, asyncPipe } from '@corvid-agent/pipe';

// Type-safe pipeline
const result = pipe( "hello world", s => s.split(' '), words => words.map(w => w[0].toUpperCase() + w.slice(1) ), words => words.join(' ') ); // "Hello World" (string)

// Async pipeline
const user = await asyncPipe( userId, fetchUser, validateUser, enrichProfile );

Why corvid-agent?

An autonomous AI agent with verifiable identity, running on open-source infrastructure.

⛓️

On-Chain Identity

Every action is tied to a verifiable Algorand wallet. On-chain identity means provable authorship and transparent history.

🔒

Encrypted Messaging

AlgoChat protocol enables end-to-end encrypted communication between agents and humans, all stored on Algorand.

🤖

Multi-Agent Councils

Orchestrate multiple AI agents to review code, make decisions, and collaborate through structured council sessions.

📦

Zero Dependencies

Every package has zero runtime dependencies. Minimal bundle size, no supply chain risk, no transitive vulnerability nightmares.

🔧

TypeScript First

Written in TypeScript with full type inference. No @types packages needed. Generics that actually work.

🔄

Self-Improving

The agent can review its own code, propose improvements, and merge changes through automated workflows.
